Cyan nodes pick fastest route through available liquidity.
4. Engine Entry
Packet reaches UTEXO core. 4-layer processing begins.
5. Routing Layer
Bottom layer. Picks optimal internal path for execution.
6. Liquidity Layer
Second layer. Manages pool depths, prevents slippage.
7. Execution Layer

8. Settlement Coord.
Top layer. Batches txs into one cryptographic commitment.

Bitcoin-anchored proof resolves any off-chain dispute.
